HTML5 is The New HTML Standard

Video courtesy of Big Buck Bunny

HTML5

  • New Elements
  • New Attributes
  • Full CSS3 Support
  • Video and Audio
  • 2D/3D Graphics
  • Local Storage
  • Local SQL Database
  • Web Applications

HTML5 Multimedia

With HTML5, playing video and audio is easier than ever.

  • HTML5 <video>
  • HTML5 <audio>

HTML5 Applications

With HTML5, web application development is easier than ever.

  • Local data storage
  • Local file access
  • Local SQL database
  • Application cache
  • Javascript workers
  • XHTMLHttpRequest 2

HTML5 Graphics

With HTML5, drawing graphics is easier than ever:

  • Using the <canvas> element
  • Using inline SVG
  • Using CSS3 2D/3D

HTML5 uses CSS3

  • New Selectors
  • New Properties
  • Animations
  • 2D/3D Transformations
  • Rounded Corners
  • Shadow Effects
  • Downloadable Fonts

Read more in our CSS3 tutorial.

Semantic Elements

New elements for headers, footers, menues, sections and articles.

HTML5 Forms
